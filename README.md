--SQL-Projekt: Relationales Shop-System--

Dieses Projekt umfasst die Planung, Strukturierung und Abfrage einer relationalen Datenbank für einen Online-Shop. Erstellt mit SQLite.

--Datenbank-Struktur--
Die Datenbank besteht aus 5 miteinander verknüpften Tabellen:

Kunden: Stammdaten der Käufer.

Produkte: Artikelkatalog mit Lagerbestand.

Lieferanten: Zuordnung der Produkte zu Herstellern.

Bestellungen: Kopfdaten (Wer hat wann bestellt?).

Bestellpositionen: Detaildaten (Welche Artikel in welcher Menge?).

"Erstellt im Rahmen vom Informatik Unterricht."
